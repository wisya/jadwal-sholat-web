Kebijakan Privasi (Privacy Policy)

Terakhir Diperbarui: 16 Juli 2026

iOnline Studio ("kami", "kita", atau "milik kami") mengoperasikan aplikasi seluler Jadwal Sholat Indonesia ("Aplikasi"). Halaman ini memberi tahu Anda tentang kebijakan kami terkait pengumpulan, penggunaan, dan pengungkapan data pribadi saat Anda menggunakan Aplikasi kami.

1. Pengumpulan dan Penggunaan Informasi

Untuk memberikan pengalaman terbaik saat menggunakan Aplikasi, kami mungkin meminta Anda untuk memberikan kami informasi tertentu yang dapat mengenali Anda secara pribadi. Namun, Aplikasi kami dirancang dengan mengutamakan privasi Anda:

  • Data Lokasi (GPS): Aplikasi ini meminta izin akses lokasi di perangkat Anda secara real-time murni untuk keperluan menghitung jadwal sholat dan arah kiblat secara astronomis. Data lokasi Anda diproses secara lokal di perangkat Anda dan TIDAK PERNAH dikirim, disimpan, atau dijual ke server kami maupun pihak ketiga manapun.

2. Layanan Pihak Ketiga

Aplikasi ini mungkin menggunakan layanan API pihak ketiga (seperti api.myquran.com atau Open-Meteo) untuk mengambil data kalender publik atau data elevasi dataran tinggi. Layanan pihak ketiga tersebut tidak mengumpulkan data pribadi Anda melalui Aplikasi kami.

3. Data Log dan Analitik

Kami tidak mengumpulkan Data Log rahasia. Jika terjadi kerusakan/crash pada aplikasi, data diagnostik anonim mungkin dikirim melalui layanan standar Google Play Services semata-mata untuk tujuan perbaikan bug, tanpa menyertakan informasi yang mengidentifikasi Anda secara personal.

4. Keamanan Data

Kami sangat menghargai kepercayaan Anda dalam memberikan akses ke lokasi perangkat Anda, oleh karena itu kami menggunakan kalkulasi offline di dalam perangkat (on-device processing). Tidak ada data sensitif yang ditransmisikan keluar dari HP Anda yang rentan disadap.

5. Perubahan pada Kebijakan Privasi Ini

Kami dapat memperbarui Kebijakan Privasi kami dari waktu ke waktu. Kami menyarankan Anda untuk meninjau halaman ini secara berkala untuk mengetahui setiap perubahan. Perubahan pada Kebijakan Privasi ini efektif saat diposting di halaman ini.
